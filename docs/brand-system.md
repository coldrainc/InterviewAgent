# Aivago Brand System

## Palette

- Ink: `#12302A` for the brand base, app chrome, and deep contrast.
- Intelligence Blue: `#2F63E8` for primary actions and active states.
- Feedback Teal: `#0F8F8F` for secondary emphasis, progress, and model feedback.
- Canvas: `#EEF4F1` and `#F7FAF8` for calm product surfaces.
- Text: `#14251F`, muted text `#64756F`.

## Icon

The icon uses a rounded-square gradient with an interview chat mark:

- `apps/desktop/public/favicon.svg`: browser favicon source.
- `apps/desktop/public/favicon-32.png`: legacy browser favicon.
- `apps/desktop/public/icon-192.png`: web manifest icon.
- `apps/desktop/public/icon-512.png`: high-resolution web/app icon.
- `apps/desktop/public/apple-touch-icon.png`: iOS home-screen icon.
- `apps/desktop/src/assets/app-icon.png`: Electron window and Dock icon.
- `apps/miniapp/miniprogram/assets/app-icon.png`: miniapp in-product brand icon.
