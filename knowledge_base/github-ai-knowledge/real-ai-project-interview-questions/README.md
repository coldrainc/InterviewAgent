# 真实 AI 项目面试题库

这组题目用于补充面试官知识库，重点覆盖真实 AI 工程项目中常被追问的内容：需求澄清、系统设计、指标、数据、模型、检索、Agent 编排、上线、成本、稳定性、安全和故障复盘。

## 使用方式

- 面试官可根据候选人的项目经历选择对应题目。
- 每道题包含“真实问法”“面试官想听什么”“可继续追问”。
- 候选人回答不应只背概念，需要讲清楚方案、取舍、指标和落地结果。

## 题目分类

- `rag-project-questions.md`：RAG / 知识库问答 / 搜索增强项目。
- `agent-project-questions.md`：Agent / 工具调用 / 多 Agent / 工作流项目。
- `llm-application-questions.md`：LLM 应用、Prompt、函数调用、结构化输出项目。
- `evaluation-production-questions.md`：评测、观测、灰度、成本、性能、部署。
- `safety-data-questions.md`：安全、权限、数据治理、Prompt Injection、隐私合规。

## 面试判断原则

高质量回答通常具备：

1. 有清晰业务场景，而不是只说“做了一个聊天机器人”。
2. 有可验证指标，如召回率、准确率、幻觉率、P95 延迟、成本、命中率、人工采纳率。
3. 能讲出失败案例和复盘，而不是只讲最终方案。
4. 能解释为什么这么选型，以及为什么没有选其他方案。
5. 能把模型能力和工程系统边界分开讲清楚。
